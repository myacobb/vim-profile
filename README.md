# Vim Profile

## Clone repository
```console
git clone https://github.com/myacobb/vim-profile.git
```

## Run setup
```console
. setup.sh
```


## Launch Vim
```console
vim
```

## Install plugins
Run the plugin installer.
```console
:PluginInstall
```

## Replace vimrc
```console
cp vimrc.jc ~/.vimrc
```
